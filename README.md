#  FitLife – Android Fitness Management App

![GitHub release](https://img.shields.io/github/v/release/mrselemogo777-gif/FitLife-Android-App)
![License](https://img.shields.io/github/license/mrselemogo777-gif/FitLife-Android-App)
![Language](https://img.shields.io/github/languages/top/mrselemogo777-gif/FitLife-Android-App)

FitLife is a native Android fitness application developed using Java and SQLite.  
The app enables users to register, log in, create and manage workouts, track completion status, and delegate workout plans via SMS.

Developed as part of Advanced Android Mobile Development

---

## Project Overview

FitLife demonstrates advanced native Android development principles:

- User authentication system
- Local data persistence with SQLite
- Workout CRUD operations
- SMS delegation feature
- Material Design-compliant UI
- Linear and intuitive navigation

The application follows a **modified MVC architecture** for maintainability and best practices.

---

##  Core Features

### User Authentication
- User registration & login
- Duplicate username prevention
- Input validation and error feedback

###  Workout Management (CRUD)
- Create, read, update, delete workouts
- Toggle workout completion status

###  SMS Delegation
- Send workout details via device SMS app
- Pre-filled message content

###  UI & UX
- Material Design 3 principles
- Responsive layouts using ConstraintLayout
- Accessible design with proper color contrast and touch targets
- Smooth activity transitions
